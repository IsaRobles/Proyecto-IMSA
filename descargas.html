INICIO COMUNIDAD DESCARGAS CONTACTO
Queremos darle las herramientas que usted necesita para que pueda manejar IMSA. Antes que nada, para poder utilizar IMSA debe de contar con lo siguiente:

  • Sistema Operativo Windows 7. (Sin importar que sea home basic, starter, professional, etc.)

  • Permisos de administrador para poder instalar el controlador de la sirena y el foco.

    • Para poder instalar el software que le permitirá monitorear sus servicios requiere de dos archivos importantes, Run-timer y VISA. El primer archivo le permite a usted poder ejecutar el arhcivo .exe IMSA, sin este el equipo le marcará un error de ejecución, si desea tener IMSA en multiples equipos debe de instalar este archivo en cada uno de ellos. El segundo archivo le permite conocer en que puerto se encuentra conectado el dispositivo que controla los indicadores audible y visual para agregarlo a la dirección a la cuál se comunicara el software con el hardware. En el Manual de Usuario viene la guía de instalación y explicación de los pasos a seguir para el uso de los archivos.

      2. Driver del hardware

      3. Run-timer

      4. VISA

      5. IMSA
